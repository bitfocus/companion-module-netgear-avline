## Netgear AVLine

Control Netgear AVLive AV-over-IP network switches.

## Configuration

- Enter the switch's management IP, username, and password in the module settings

## Actions

- Set POE
- Power Cycle POE
- Reboot Switch

## Feedback

- POE Enabled
- Link Status
- Reboot Switch

## Variables

- active_ports
- cpu_usage
- memory_usage
- uptime

_Per Port Variables_

- poe_current_power
- poe_status
